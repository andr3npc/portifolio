# Certificates

Upload your certificate files here (images or PDFs). They'll be wired
into the site once they're in place.

Guidance:

- **Format:** JPG/PNG/WebP for images, or PDF. Images display inline;
  PDFs can be linked/opened.
- **Names:** short, lowercase, hyphenated, and descriptive so they sort
  and label cleanly, e.g. `ibm-mainframe-developer.png`,
  `coursera-agile-jira.png`. Avoid spaces.
- **Size:** keep each under ~500 KB (resize scans/screenshots) so the
  page stays fast.
- Files in `public/` are copied to the site root at build time, so
  `public/certificates/ibm-mainframe-developer.png` is served at
  `/certificates/ibm-mainframe-developer.png` (base path applied
  automatically).

Once uploaded, tell me and I'll build a certificates gallery (or link
them from the existing Certifications list) — your call on the layout.
